# 💬 ChatApp — Real-Time Messaging Platform

A full-stack real-time chat application built with **Django, WebSockets, JWT authentication**, and a modern WhatsApp-style interface.  
Supports direct messaging, chat rooms, friend requests, and real-time presence.

---

## 🚀 Live Demo

👉 https://realtime-chat-app-zx0u.onrender.com

---

## ✨ Features

- 🔐 JWT Authentication (Login / Register)
- 📧 Email OTP Verification
- 💬 Real-Time Chat (WebSockets)
- 👤 Direct Messaging (1-to-1)
- 👥 Group Chat Rooms
- 🤝 Friend Request System
- 🟢 Online / Offline Presence
- 🔔 Unread Message Counters
- 📎 File Attachments
- 🖼️ Profile Picture Upload
- 🔑 Forgot Password (Email Reset)
- ⚡ Redis + Celery Background Tasks
- 🐳 Docker Support
- 🌐 Deployed on Render
